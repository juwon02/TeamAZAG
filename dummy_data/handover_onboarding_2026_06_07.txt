===SECTION: 핵심 기능 요약===
1. 파일 업로드 & AI 분석
   - 지원 형식: .txt, .pdf, .docx, .csv
   - 업로드 후 자동 AI 분석 (평균 8~30초)
   - Todo, 이슈 자동 추출

2. Todo 관리
   - AI 제안 / 진행 중 / 완료 탭 분리
   - 승인(approve) / 반려(reject) / 완료(done) 액션
   - confidence_score, priority 표시

3. 이슈 로그
   - High/Medium/Low severity 분류
   - 출처 문서 추적

4. Dashboard
   - 전체 현황 카드 (Todo 수, 이슈 수, 완료율)
   - 주간 추이 차트

5. AI Assistant
   - 문서 기반 Q&A (FAISS RAG)
   - 출처 파일명 표시
   - 업무 무관 질문 자동 차단

6. 보고서 생성
   - 주간/월간 보고서 자동 생성
   - 날짜 범위 선택 후 1클릭 생성
